# Linux.do SidePeek

给 `https://linux.do` 的主题列表页加一个右侧抽屉预览：点击帖子标题后，不离开列表页，右边直接展开内容。

![Linux.do SidePeek 抽屉预览](assets/post/drawer-preview.png)

## 当前功能

- 拦截主题列表中的标题点击，改为右侧抽屉预览
- 优先通过 Discourse 的 topic JSON 接口渲染内容
- 接口不可用时，自动回退为右侧 iframe 整页预览
- 支持右上角 `选项` 按钮弹出临时设置窗口
- 设置窗口支持切换：`预览模式`、`内容范围`、`回复排序`、`抽屉宽度`
- 设置窗口支持 `关闭` 按钮、点击空白区域关闭、`Esc` 关闭
- 设置修改后会自动保存到本地，下次打开继续生效
- 支持拖动抽屉左边缘来自定义宽度（窄屏下可能不可用）
- 支持 `上一帖` / `下一帖` 快速浏览当前列表
- 支持 `Alt + ↑ / Alt + ↓` 快捷切换上一帖 / 下一帖
- 支持 `最新回复优先`：保留首帖在顶部，后续回复按从新到旧显示
- 每次页面加载后首次打开时会显示一次预览说明，提醒抽屉视图与原帖页可能略有差异
- 标签对象会自动提取可读名称，避免显示 `[object Object]`

## 使用方式

1. 打开主题列表页，例如 `https://linux.do/` 或 `https://linux.do/latest`
2. 点击任意主题标题，右侧抽屉会展开
3. 如需调整预览行为，点击右上角 `选项`
4. 如需完整原帖页，点击右上角 `新标签打开`
5. 如需关闭抽屉，按 `Esc` 或点击右上角 `关闭`

## 预览选项

- `预览模式`
  - `智能预览`：优先使用 JSON 渲染内容
  - `整页模式`：直接以内嵌网页方式显示原帖页
- `内容范围`
  - `完整主题`
  - `仅首帖`
- `回复排序`
  - `默认顺序`
  - `最新回复优先`
- `抽屉宽度`
  - `窄` / `中` / `宽` / `自定义`
  - 也可以直接拖动抽屉左边缘调整，窄屏下可能不可用
- `恢复默认`
  - 一键恢复为默认设置

## 已知说明

- 抽屉预览是便捷阅读视图，不保证和原帖完整页面 100% 一致
- 某些帖子里的外链资源、嵌入内容、动态组件，在抽屉里可能和原帖页表现不同
- 如果需要最完整的展示效果，请使用 `新标签打开`
