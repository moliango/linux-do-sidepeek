# 更新日志

这个文件只记录会影响实际使用体验的变化。

## [0.5.0] - 2026-03-20

### 新增
- 支持抽屉模式设置；现在可以在 `挤压模式` 和 `浮层模式` 之间切换，浮层模式下支持点击抽屉外区域直接关闭。
- 支持在“首帖 + 最新回复”视图下手动刷新最新回复，刷新时会尽量保留当前抽屉滚动位置，不必关闭后重新打开主题。
- 支持在抽屉快速回复框里直接粘贴图片自动上传，成功后会插入站点原生的 `upload://` Markdown。
- GitHub Release 现在会同时产出 Chrome ZIP 和 Firefox 未签名 XPI，方便在 Firefox 里做临时安装和验证。

### 改进
- 列表页“查看 x 个新的或更新的话题”提示条会固定在中间栏顶部区域，点击后回顶刷新更稳定。
- 抽屉内为楼主回复增加 `Topic Owner` 标记，阅读长帖时更容易识别楼主发言。
- 修复浮层模式下抽屉打开时点击另一个列表主题会先收起、第二次才打开的问题；现在会直接切换到新主题。

### 致谢
- 感谢 PR 贡献者 `shengmingboai` 带来了抽屉模式设置（挤压模式 / 浮层模式）。
- 感谢 PR 贡献者 `majorcheng` 带来了新话题提示条交互修复、`Topic Owner` 标记和最新回复刷新按钮。

## [0.4.0] - 2026-03-15

### 新增
- 支持在抽屉内直接回复主题或指定楼层；点击帖子卡片下方「回复这条」可以针对性回复，点击浮动「回复」按钮可以回复整个主题。支持 Markdown，`Ctrl+Enter` / `Cmd+Enter` 快捷发送。
- 支持无限滚动自动加载更多帖子；在"完整主题 + 默认顺序"模式下，下滑接近底部会自动拉取后续回复，底部显示加载进度。
- 加载更多时保持当前滚动位置，不会跳回顶部。
- 发送回复成功后，新帖子会自动合入当前视图并滚动定位到新帖。

### 改进
- 合并帖子流（post stream）时同步合并 `stream` 数组和 `posts_count`，为增量加载提供完整索引。
- 主题 URL 归一化改用 `parseTopicPath` 解析，路径截取更准确。
- 设置加载改为逐字段校验白名单，避免旧存储里的脏值污染新设置。
- JSON 预览失败时自动回退为 iframe 整页预览，不再显示纯文案错误。

### 致谢
- 感谢 linux.do 群友 `zhoudashuai` 贡献了抽屉内回复和无限滚动加载的初始实现，本版本基于其提交的改动合并而来。

## [0.3.0] - 2026-03-10

### 新增
- 支持抽屉内图片点击全屏放大预览；点击遮罩层、关闭按钮或按 `Esc` 可退出，不再默认跳到独立图片页。

### 修复
- 修复复用同一个抽屉切换到其他主题时，旧图片预览遮罩可能残留在新内容上方的问题。
- 修复带普通外链的图片在抽屉里被误拦截的问题；现在只有图片本身或其原图链接会触发抽屉内预览，普通链接仍按原目标打开。

### 致谢
- 感谢 linux.do 群友 `zhoudashuai` 提出“侧边栏图片点击后改成抽屉内全屏放大、点击遮罩层退出”的优化建议，并完成了这次体验补齐与相关修复。

## [0.2.0] - 2026-03-08

### 修复
- 修复“侧边预览不计浏览”的问题；现在智能预览和整页模式都会尽量按站点正常进帖的方式触发论坛计数。
- 修复切换智能预览 / 整页模式时可能重复计浏览的问题；现在同一次抽屉会话里，同一主题不会因为切模式被重复计数。
- 修复部分页面“右侧滑窗不弹出”的问题；现在首页、搜索页、分类页、标签页、用户发帖流等更多带主题链接的页面都可以直接侧边预览。
- 修复主题详情页里的推荐主题等标题链接不能侧边预览的问题；现在只排除正文、编辑器和弹层里的链接，不会误伤正常主题标题。
- 修复某些场景下误触发右侧滑窗的问题；帖子正文、编辑器、弹层菜单等区域里的主题链接默认不会被接管。
- 修复未读帖、搜索命中等链接总是从首帖打开的问题；现在会尽量保留你点击时对应的楼层位置。
- 修复深链接在“仅首帖”或“首帖 + 最新回复”模式下内容错位的问题；现在不会再把某条回复误当成首帖固定在顶部。
- 修复 `/last`、未读定位等 helper 链接在智能预览里丢失目标位置的问题；现在会尽量保留你点击时的最后位置或目标楼层。
- 修复部分主题链接识别错乱的问题；短格式楼层链接、搜索结果、用户动态里的同主题多条记录，现在不会再跳错帖或让“上一帖/下一帖”定位错乱。

### 体验调整
- 调整回复排序的产品定义；智能预览默认仍是正常顺序，“首帖 + 最新回复”现在明确表示优先展示最新一批回复，不再暗示整帖完整倒序。
- 调整重复点击同一帖子的处理方式；扩展不再自己做长时间去重，是否计入浏览交给论坛本身判断。
- 调整右侧滑窗的适用范围说明；现在强调的是“支持站内可预览的主题链接”，不再只写传统列表页。

## [0.1.0] - 2026-03-07

### 首次发布
- 支持在 Linux.do 右侧滑窗预览帖子，不用频繁进出帖子页。
- 支持智能预览与整页模式切换。
- 支持上一帖、下一帖和快捷键切换。
- 支持预览选项、抽屉宽度调整与设置保存。
